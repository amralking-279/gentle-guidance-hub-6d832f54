## تحسينات قالب الفجر

### 1. شريط التبويبات السفلي (Bottom Tab Bar)
إعادة تصميم `PrayerNowBottomTab` ليطابق الصور المرفقة:
- 5 تبويبات: استكشف / الصلاة / المزيد / الأذكار / القرآن
- خلفية فاتحة كريمية تتناغم مع قالب الفجر (بدل الأخضر الداكن الحالي)
- أيقونات Lucide حديثة بحدود نظيفة (Compass, Bell, LayoutGrid, Sparkles, BookOpen)
- التبويب النشط: لون ذهبي/إيمرالد بارز + نقطة سفلية صغيرة
- التبويبات غير النشطة: لون إيمرالد باهت
- خط Cairo bold وحجم صغير أنيق
- ظل علوي ناعم وحد علوي ذهبي رفيع

### 2. زر تغيير الثيمات الواضح
إضافة زر عائم/ثابت في كل صفحة (داخل قالب الفجر فقط) يفتح صفحة `/more/themes`:
- موقع ثابت أعلى التاب بار السفلي (يمين الشاشة)
- شكل دائري بحدود ذهبية وأيقونة `Palette` من Lucide
- لون كريمي شبه شفاف مع backdrop-blur
- تلميح "تغيير الشكل" عند الضغط
- يظهر في كل الصفحات تحت قالب الفجر

### 3. ضبط الألوان لتتناغم مع البيج الفاتح
تحديث CSS قالب الفجر في `src/styles.css`:

| العنصر | اللون الجديد |
|---|---|
| الخلفية | `linear-gradient(180deg, #faf3e3 0%, #f0e2c4 100%)` (موجود — تثبيت) |
| النص الأساسي | `#2d4a3a` (إيمرالد داكن دافئ) |
| النص الثانوي | `#6b8472` |
| الكروت | `#fffdf5` بحدود `#d4b885` خفيفة |
| اللون الذهبي للأكسنت | `#c9a449` |
| الإيمرالد الأساسي للأزرار | `#1f6b4a` |
| حدود | `rgba(180, 138, 70, 0.18)` |
| ظلال | دافئة بلون ذهبي خفيف بدل الأسود |

تخصيص إضافي:
- الأزرار الإيمرالد تصبح بتدرّج `#1f6b4a → #2d8a5f` مع نص كريمي
- المفاتيح والـ chips تصبح بخلفية كريمية وحدود ذهبية
- الفواصل والـ dividers بلون ذهبي شفاف
- أيقونات الـ Navbar/Footer المخفية لا تحتاج لمسات إضافية

### 4. تحديث الأيقونات الحديثة
استبدال الأيقونات في `PrayerNowBottomTab` و `PrayerNowHome` بأيقونات Lucide أحدث وأوضح:
- استكشف → `Sparkles` (نشاط)
- الصلاة → `Clock` بدل `Bell`
- المزيد → `LayoutGrid` بدل `Grid3x3`
- الأذكار → `HeartHandshake` أو `Sparkle`
- القرآن → `BookOpenText` بدل `BookOpen`
- زر الثيمات → `Palette`
- إعدادات → `Settings2`

### الملفات المتأثرة

```text
src/components/templates/PrayerNowBottomTab.tsx   إعادة تصميم كاملة (ألوان فاتحة + أيقونات جديدة)
src/components/ui-custom/ThemeQuickButton.tsx     جديد — زر عائم لفتح صفحة الثيمات
src/routes/__root.tsx                              تركيب الزر العائم عند تفعيل قالب الفجر
src/styles.css                                     ضبط ألوان html.app-template-prayer-now
src/components/templates/PrayerNowHome.tsx        تحديث الأيقونات وتنسيق الألوان مع الباليت الجديدة
```

### ملاحظات تقنية

- الزر العائم سيُركّب في `RootComponent` بشرط `template === "prayer-now"` (مثل التاب بار)
- موضع الزر: `fixed bottom-20 right-4 z-50` فوق التاب بار مباشرة
- التاب بار يأخذ `bg-[#fffaf0]/95` مع `backdrop-blur-xl` للحفاظ على إحساس زجاجي فاتح
- الـ active state يستخدم `motion.div` صغيرة كنقطة سفلية بدل تغيير لون كامل
- الـ safe-area-inset يبقى كما هو للأجهزة المحمولة
